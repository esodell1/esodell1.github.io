Ball Balancing Plane

Ball Balancing System

This project was design using a standard 720p USB web cam plugged into an Intel Edison development kit. Using OpenCV computer vision software, the Edison tracks the position of a colored ball in an XY plane, and using a software encoded PID controller, translates the resultant controller output to servo positions that control the platform on which the ball is moving.

The project was successfully able to center the ball after a minimal frictional resistance was added to the plane.

Animatronic Hand

Remotely Operated Animatronic Hand

This project consists of a (cheaply) fabricated hand with five fingers. The fingers are locally controlled by individual servo motors mounted to the hand apparatus. This is controlled by the Intel Edison development board, which remotely connects to a separate Intel Edison board. The second board receives input from a glove that is wired with flexible variable resistance strips, determining the position of the wearer's fingers, and sending that information to the hand apparatus.

The project was successfully able to mimic the hand of the glove wearer accurately, and with minimal delay due to network transmissions.

Sleep Monitor

SleepSafe Wearable Heart Rate/Pulse Oximeter

This is an ongoing project untaken at the request of the Children's Therapy Unit of the Multicare Good Samaritan Hospital in Puyallup, WA. The purpose of this device is to monitor the heart rate and peripheral blood oxygen saturation in patients with potential respiratory issues.

The device uses infrared and red LEDs to measure the expansion and contraction of venous tissue, and from that, determine an approximate heart rate. Doing this simultaneously with red and infrared allows for the contrasting of the different wavelengths to calculate the peripheral blood oxygen saturation.

This information will be sent to an Android smartphone, where it will be viewed in near real time, and allowing for storage of history as well as actuating alarms based on predetermined conditions, such as high heart rate or low oxygen saturation.
